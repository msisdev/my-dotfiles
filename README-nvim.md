# /nvim

## [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim?tab=readme-ov-file)
General guide on how to start using `kickstart.nvim`
### What is this?
`kickstart.nvim` features:
1. A minimal nvim config
2. A bunch of guides

### What should I do?
1. Clone
```
> git clone https://github.com/nvim-lua/kickstart.nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```
2. Run
```
> nvim
```

### What's next?
Watch [youtube](https://www.youtube.com/watch?v=m8C0Cq9Uv9o&feature=youtu.be)

Run `:Tutor` (in neovim)

Run `:help`

Search things by `[Space] > [S]`
